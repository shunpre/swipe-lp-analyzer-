# 配布・販売ガイド

## 📦 このプロジェクトを他の人に販売する方法

### ✅ 環境への影響について

**結論: 他の人の環境に悪影響を与えることはありません。**

このプロジェクトは**独立したPythonアプリケーション**であり、以下の理由から安全に配布できます：

#### 1. **仮想環境（venv）の使用を推奨**
購入者は自分の仮想環境内でインストールするため、システム全体には影響しません。

```bash
# 購入者が実行する手順
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
streamlit run app/main_v2.py
```

#### 2. **依存ライブラリは標準的なもののみ**
`requirements.txt`に含まれるライブラリは、全て安全で広く使用されているものです：

- `streamlit` - Webアプリフレームワーク
- `pandas` - データ分析
- `plotly` - グラフ可視化
- `requests` - HTTP通信
- `beautifulsoup4` - HTMLパース
- `Pillow` - 画像処理
- `openai` - OpenAI API（オプション）
- `python-dotenv` - 環境変数管理

#### 3. **システムファイルを変更しない**
このアプリケーションは：
- ❌ システム設定を変更しない
- ❌ レジストリを変更しない（Windows）
- ❌ グローバルな環境変数を変更しない
- ✅ 指定されたディレクトリ内でのみ動作

## 📋 配布パッケージの準備

### 方法1: GitHubリポジトリとして配布（推奨）

**メリット:**
- ✅ 更新が簡単
- ✅ バージョン管理が可能
- ✅ プロフェッショナルな印象

**手順:**
1. GitHubリポジトリを作成（プライベートまたはパブリック）
2. 購入者にリポジトリへのアクセス権を付与
3. 購入者は`git clone`でダウンロード

```bash
# 購入者が実行
git clone https://github.com/shunpre/swipe-lp-analyzer-.git
cd swipe-lp-analyzer-
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app/main_v2.py
```

### 方法2: ZIPファイルとして配布

**メリット:**
- ✅ シンプル
- ✅ Gitの知識不要

**手順:**
1. プロジェクトをZIPファイルに圧縮
2. 購入者にZIPファイルを送信（メール、Dropbox、Google Drive等）
3. 購入者は解凍して実行

## 📝 購入者向けREADMEの作成

以下のような`README.md`を用意することをお勧めします：

```markdown
# スワイプLP分析ツール

## 🚀 インストール方法

### 必要な環境
- Python 3.8以上
- pip

### 手順

1. **仮想環境を作成**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

2. **依存ライブラリをインストール**
   ```bash
   pip install -r requirements.txt
   ```

3. **アプリを起動**
   ```bash
   streamlit run app/main_v2.py
   ```

4. **ブラウザで開く**
   - 自動的にブラウザが開きます
   - または http://localhost:8501 にアクセス

## 📊 使い方

1. サイドバーからLPを選択
2. 期間を選択
3. 各タブで分析結果を確認

## 🔧 トラブルシューティング

### エラー: ModuleNotFoundError
```bash
pip install -r requirements.txt
```

### ポート8501が使用中
```bash
streamlit run app/main_v2.py --server.port=8502
```

## 📞 サポート

問題が発生した場合は、[あなたの連絡先]までご連絡ください。
```

## 🔒 セキュリティとプライバシー

### 配布前に確認すること

1. **APIキーやパスワードが含まれていないか**
   ```bash
   # .envファイルは配布しない
   # .env.exampleのみ配布
   ```

2. **個人情報が含まれていないか**
